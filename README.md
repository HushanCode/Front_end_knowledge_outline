## 关于仓库

这个仓库是本人平时学习生活中的学习笔记，主要是总结前端常见的知识点，在平时学习的过程中我更想知道为什么，特别是对于八股文，我更喜欢通过理解实践的方法去记忆，单纯的靠背是很难长久的记忆的，希望也能对大家有所帮助。
